phpMyAdmin

Import File SQL ke Database

  1. 1. Buka phpMyAdmin.
  2. 2. Klik database yang dibuat.
  3. 3. Klik tab Import.
  4. 4. Pilih file SQL.
  5. 5. Klik Go atau Import.
  6. 6. Tunggu pesan sukses.
Tanda Berhasil

Tabel settings, products, orders, members, coupons, coupon_products, coupon_redemptions, dan bot_user_states muncul.

Jika import gagal

• Pastikan database benar.

• Pastikan file SQL utuh.

• Jangan import dua kali ke database yang sama.

• Gunakan database baru jika ingin mengulang.

5
File Manager

Upload Seluruh File Script PHP ke Hosting

Upload paket source code iMersTeleship ke folder domain atau subdomain yang akan digunakan. Setelah file ZIP berhasil diunggah, ekstrak seluruh isinya di folder tersebut.

  1. 1. Buka File Manager di cPanel.
  2. 2. Masuk ke folder domain atau subdomain tujuan.
  3. 3. Upload file ZIP source code iMersTeleship.
  4. 4. Klik kanan file ZIP, lalu pilih Extract.
  5. 5. Pastikan semua file berada langsung di folder tujuan dan tidak masuk ke folder ganda.
Contoh struktur file yang benar:

public_html/
├── .htaccess
├── admin.php
├── config.php
├── db.php
├── index.php
├── install.php
├── Telegram.php
└── webhook.php
Domain Utama

Jika dipasang di domain utama, file biasanya berada langsung di dalam public_html.

Subfolder

Jika dipasang di folder teleship, lokasinya menjadi public_html/teleship/.

Jangan menghapus install.php sebelum proses instalasi selesai. File tersebut digunakan untuk mengisi koneksi database dan Base URL secara otomatis.
6
Installer Resmi

Buka install.php dan Isi Konfigurasi Database

Anda tidak perlu membuka atau mengedit kode koneksi database secara manual. Jalankan installer melalui browser, lalu isi formulir sesuai data database yang sudah dibuat di cPanel. 

https://domainanda.com/install.php

Jika script dipasang di subfolder, gunakan URL lengkap seperti: https://domainanda.com/teleship/install.php

1. Database Host

Biasanya menggunakan localhost, kecuali hosting Anda memberikan host database yang berbeda.

2. Database Name

Masukkan nama database lengkap sesuai yang tampil di cPanel, termasuk prefix akun hosting.

3. Database User

Masukkan username database lengkap yang sudah dihubungkan ke database dengan hak akses ALL PRIVILEGES.

4. Database Password

Masukkan password user database yang dibuat sebelumnya.

5. Base URL Website

Masukkan URL folder tempat source code diunggah. Contoh: https://domainanda.com/teleship/

Jalankan Proses Instalasi
  1. Periksa kembali seluruh data yang dimasukkan.
  2. Klik tombol Install Sekarang.
  3. Tunggu sampai installer menampilkan informasi bahwa konfigurasi berhasil disimpan.
  4. Installer akan mengisi koneksi database dan Base URL ke file konfigurasi secara otomatis.
Installer hanya mengatur koneksi database dan Base URL. Struktur tabel database tetap berasal dari file SQL master yang sudah Anda import melalui phpMyAdmin pada langkah sebelumnya.
Jika installer gagal menyimpan konfigurasi

• Periksa kembali nama database, username, password, dan hak akses user database.

• Pastikan file config.php tersedia di folder yang sama dengan install.php.

• Jika ada pesan file tidak dapat ditulis, periksa permission file melalui File Manager. Setelah instalasi berhasil, kembalikan permission ke pengaturan aman hosting.

• Pastikan Base URL menggunakan HTTPS dan mengarah ke folder instalasi yang benar.

Setelah instalasi berhasil dan dashboard dapat dibuka, hapus atau ganti nama install.php jika installer tidak menguncinya secara otomatis. Langkah ini mencegah orang lain menjalankan ulang installer.

Troubleshooting

Masalah yang Sering Terjadi

Error database

Periksa host, nama database, username, password, dan ALL PRIVILEGES.

Bot tidak merespons /start

Periksa token, HTTPS, URL webhook, dan getWebhookInfo.

Error 404 pada webhook.php

Pastikan path file dan URL sama.

Error 500 atau halaman putih

Periksa error_log, versi PHP, sintaks, dan kelengkapan file.

Import SQL gagal karena tabel sudah ada

Gunakan database baru atau pulihkan backup sebelum mengulang.
