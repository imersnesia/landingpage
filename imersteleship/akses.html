Quick Start

Urutan Instalasi yang Direkomendasikan

Pastikan Anda mengikuti tutorial dan melakukan backup sebelum mengubah source code atau struktur database.

1

Buat Database Baru

Buat database dan user database melalui cPanel, kemudian berikan seluruh hak akses yang diperlukan.

2

Import File SQL

Buka phpMyAdmin, pilih database, lalu import atau jalankan isi file SQL yang sudah disediakan.

3

Upload Source Code

Ekstrak dan unggah semua file PHP ke domain atau subdomain yang akan digunakan.

4

Atur Database & Bot Telegram

Masukkan detail koneksi database, Bot Token, Admin Chat ID, URL webhook, dan metode pembayaran.

5

Lakukan Pengujian

Tes katalog, checkout, kupon, pembayaran, approval order, dan akses produk sebelum digunakan secara publik.

Backup Sebelum Modifikasi

Simpan salinan source code dan database asli sebelum mengedit file, menambah fitur, atau mengubah struktur tabel.

Gunakan Grup Support Secara Tepat

Saat bertanya, sertakan screenshot error, nama file terkait, langkah yang sudah dilakukan, serta versi PHP atau database yang digunakan.
