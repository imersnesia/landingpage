Pertanyaan Umum

Apakah ini masih pakai Google Apps Script?

Tidak. Versi ini berbasis script PHP yang diupload ke hosting cPanel dan menggunakan webhook PHP untuk terhubung dengan Telegram Bot.

Apakah butuh hosting?

Ya. Anda membutuhkan hosting cPanel atau server PHP yang mendukung file PHP dan akses database sesuai paket script.

Apakah bisa pakai brand sendiri?

Bisa. Anda bisa mengganti nama bot, logo, domain, copywriting pesan, produk, harga, dan tampilan sales page sesuai brand sendiri.
