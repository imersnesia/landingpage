Apakah ini memakai script PHP?

Ya. Sistem berbasis script PHP yang diunggah ke hosting cPanel dan menggunakan webhook PHP untuk terhubung dengan Telegram Bot.

Apakah butuh hosting?

Ya. Anda membutuhkan hosting cPanel atau server PHP yang mendukung database MySQL atau MariaDB, HTTPS, cURL, dan webhook Telegram.

Bagaimana sistem kuponnya bekerja?

Admin membuat kupon dari dashboard. Saat checkout, pembeli menekan tombol Pakai Kupon dan memasukkan kode. Bot memvalidasi aturan promo, menghitung diskon, lalu membuat invoice berdasarkan total akhir.

Apakah kupon bisa dibatasi untuk produk tertentu?

Bisa. Kupon dapat berlaku untuk semua produk atau hanya produk pilihan, serta dapat dibatasi berdasarkan kuota, minimum transaksi, tanggal aktif, dan jumlah pemakaian per user.

Apa fungsi pagination pada dashboard?

Pagination membagi daftar pesanan, produk, dan kupon ke beberapa halaman agar dashboard tidak memanjang terus ke bawah dan tetap mudah digunakan ketika datanya sudah banyak.

Bagaimana stock digital bekerja?

Admin dapat memilih mode Unlimited, Quantity, atau Unique Item. Saat invoice dibuat, stock direservasi. Setelah pembayaran di-approve, stock menjadi terjual. Jika order ditolak, dibatalkan, atau terkena auto-cancel, stock dikembalikan menjadi tersedia.

Apakah bisa menjual akun sosial media dan voucher game?

Bisa. Gunakan mode Unique Item untuk memasukkan akun, kode voucher, serial key, PIN, atau token satu per baris. Sistem akan mengambil satu item tersedia untuk setiap order.

Apakah akun atau voucher dikirim otomatis?

Ya. Setelah admin memastikan dana masuk dan menekan Approve, item unik berubah menjadi SOLD dan data akses dikirim otomatis kepada pembeli melalui bot Telegram.

Apa yang terjadi jika order dibatalkan?

Stock yang sebelumnya berstatus RESERVED akan dilepas kembali menjadi AVAILABLE agar dapat dibeli oleh pengguna lain.

Apakah source code akan mendapatkan update berikutnya?

Tidak. Seluruh pembelian bersifat final dan tidak tersedia refund dengan alasan apa pun karena produk berupa source code digital yang dapat langsung diakses, disalin, dan digunakan. Pastikan Anda telah memeriksa demo, spesifikasi fitur, kebutuhan server, serta syarat lisensi sebelum membeli.
