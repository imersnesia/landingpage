Checklist Install

1

Upload file aplikasi ke hosting atau subdomain.

2

Buat database MySQL kosong di cPanel.

3

Upload database.sql di root aplikasi bersama install.php.

4

Buka domain/install.php lalu isi data database dan admin awal.

5

Login admin, setting toko, logo, WhatsApp, RajaOngkir, produk, dan template pesan.

Buka Panduan Lengkap

Agency License Generator

Khusus member Agency Starter dan Agency Premium. Silakan login menggunakan email dan password terdaftar untuk generate lisensi client.

Informasi Paket Agency

  • Agency Starter: 25 lisensi + bonus 5 lisensi = total 30 lisensi.
  • Agency Premium: 100 lisensi + bonus 5 lisensi = total 105 lisensi.
  • 1 lisensi digunakan untuk 1 instalasi/project toko client.
  • Generate lisensi hanya bisa dilakukan oleh akun agency aktif.

Login Agency

Gunakan email dan password terdaftar.

Belum punya akses agency? Upgrade ke paket Agency Starter atau Agency Premium untuk mendapatkan akun generate lisensi.
