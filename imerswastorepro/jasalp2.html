FAQ

Pertanyaan Umum

Apakah paket ini termasuk lisensi iMersWAStore?

Ya, setiap paket sudah termasuk 1 lisensi iMersWAStore untuk 1 instalasi/project toko.

Apakah domain pasti .com atau .id?

Untuk paket Full Setup Premium, pilihan domain .com atau .id mengikuti ketersediaan nama domain. Bila nama yang diminta tidak tersedia, Anda perlu menyiapkan alternatif nama.

Apa bedanya Full Setup Hemat?

Full Setup Hemat tetap mendapatkan domain dan hosting 1 tahun, tetapi pilihan domain dibatasi pada .my.id atau .web.id.

Apakah bisa pakai hosting sendiri?

Bisa. Pilih paket Jasa Install Only. Anda perlu menyediakan akses cPanel/panel hosting dan akses domain/DNS bila diperlukan.

Apakah termasuk input semua produk?

Paket ini fokus pada instalasi dan konfigurasi dasar. Input produk awal bisa disesuaikan dengan kesepakatan jumlah produk. Input produk banyak dapat dikenakan biaya tambahan.
